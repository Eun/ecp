
Usage
-----
    usage: ecp [OPTION] SOURCE... DESTINATION
    
    OPTION can be:
     -h=HASHALGO      decide which hash algorithm should be used.
                        HASHALGO can be:
                          NONE, MD5, SHA1, SHA224, SHA256, SHA384, SHA512.
                        Default is MD5
     -d               create empty directorys
